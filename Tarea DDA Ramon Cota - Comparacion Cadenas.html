Para un alfabeto de dos smbolos ($0,1$) ($M=2$), secuencias de longitud $L = 20$ y $R = [1,2,3,4,5,..., 15]$ calcular $P_m$ en funcin de $R$ para dos casos:

  • Con desplazamiento
  • Sin desplazamiento

Para cada valor $R$, llevar a cabo $K=30$ experimentos $MC$:

  • Sean $S_0$ y $S_1$ dos secuencias de longitur $L$ generada al azar a partir de los $M$ smbolos en el alfabeto. Comparar Existe una secuencia de $R$ posiciones que coincidan?

La probabilidad $P_M(R)$ es el nmero de secuencias para las cules la respuesta a la pregunta es verdadera dividida entre $K$.

    :param list1: Debe ser una lista con nicamente valores numricos.
    :return: unique_list: Una lista de nicamente los valores repetidos.
    """
    list_set = set(list1)
    unique_list = (list(list_set))
    return unique_list

def monte_carlo(montecarlo):
    """
    Esta funcin repite los experimentos las veces que la variable montecarlo indique.

    Se comienzan por 2 listas vacas que se llenarn con el valor de R, es decir, el nmero de posiciones contiguas en las que coinciden las secuencias secuence_0 y secuence_1. Para ello se hace primero un barrido de las secuencias ndice(s) por ndice(s). Es decir, se compararn secuence_0[R] y secuence[R]. Posterior a esto se guardan las R en que coindicen y se realiza la comparacin pero ahora con R desplazada. Es decir, secuence_0[R] con secuence_1[R+i], siendo i cualquier valor entre 0 y 20 en el menor valor de R y 0 y 5 en el mayor valor de R.
    :param montecarlo: Nmero de veces que se repetir el experimento.
    :return: listas ordenadas de R coincidentes.
    """
    lista_sm = []
    lista_cm = []
    # montecarlo = 30
    for l in range(0,montecarlo):
        secuence_0 = []
        for i in range(0,20):
            secuence_0.append(random.randint(0,1))

        secuence_1 = []
        for i in range(0,20):
            secuence_1.append(random.randint(0,1))

        ########
        lista2 = []
        for i in range(0,20):
            for j in range(i+1,21):
                if secuence_0[i:j] == secuence_1[i:j]:
                    lista2.append(len(secuence_0[i:j]))
                    lista2.sort()
        # lista_sm.append(max(lista2))
        lista_sm.append(unique(lista2))
        lista_sm.sort()
        ########
        lista3 = []
        for i in range(1,16):
            for j in range(0,21-i):
                for k in range(0,21-i):
                    if secuence_0[j:j+i] == secuence_1[k:k+i]:
                        lista3.append(len(secuence_1[k:k+i]))
        # lista_cm.append(max(lista3))
        lista_cm.append(unique(lista3))
        lista_cm.sort()
    return lista_sm, lista_cm

def list_hist(lista_de_listas, montecarlo):
    """
    Al ser una lista de listas, debemos extraer cada lista externa para hacer una lista total de todas las R coincidentes.
    :param lista_de_listas:
    :return: lista de elementos de la lista.
    """
    lista = []
    for r in range(0,montecarlo):
        for p in range(0,len(lista_de_listas[r])):
            lista.append(lista_de_listas[r][p])
    lista.sort()
    return lista

def tab_plt_hist(lista_cm,montecarlo,Title):
    """
    Funcin para crear el DataFrame e imprimir tanto la tabla como el histograma.
    :param lista_cm:lista de listas
    :param montecarlo: nmero de veces que se realizaron.
    :return:DataFrame
    """
    list_counter = Counter(list_hist(lista_cm,montecarlo))
    df = pd.DataFrame.from_dict(list_counter, orient = 'index')
    print(tabulate(df,headers=['R', 'frecuencia'],tablefmt="presto"))
    df.plot(kind='bar', figsize=(12, 7),  fontsize=15,title = Title + " " + str(montecarlo) + " veces" )
    plt.legend('Frecuencia')
    plt.xlabel('R')
    plt.ylabel('Frecuencia')
    return df

def porcentaje_hist(df, montecarlo,  Title):
    """
    Funcin para plotear el histograma con los porcentajes para cada R.
    """
    lista = []
    suma_total = montecarlo
    for i in (df[0]):
        lista.append(i/suma_total)
    df['probabilidad'] = lista
    df.plot(kind='bar', y = 'probabilidad', figsize=(12, 7),  fontsize=15, facecolor='darkgoldenrod')
    plt.title(Title + " " + str(montecarlo) + " veces")
